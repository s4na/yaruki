## 📌 主要な学習ポイント

### 7つのUX心理学原則

1. **視覚的階層（Visual Hierarchy）** 🎯
   - 情報の優先順位を視覚的に表現
   - → 迷いが減る

2. **ユーザーデライト（User Delight）** ✨
   - 予期しない喜びを生み出す
   - → ポジティブな記憶が残る

3. **認知負荷の削減（Cognitive Load）** 🧠
   - 判断の負担を減らす
   - → 完了率がUP

4. **ピーク・エンド法則（Peak-End Rule）** 🏔️
   - 最高の瞬間と終わりの瞬間が重要
   - → リピート率がUP

5. **段階的要請（Foot in the Door）** 🚪
   - 小さな要求から始める
   - → 完了しやすい

6. **ナッジ効果（Nudge）** 💨
   - 優しく行動を誘導する
   - → 強制感がない

7. **ドハティの閾値（0.4秒の壁）** ⚡
   - 0.4秒以内の反応が重要
   - → 快適さがUP

## 💡 よくある質問

### Q. 全ての原則を適用する必要があるか？
**A.** いいえ。プロジェクトに応じて、必要な原則を選んで適用してください。

**推奨**:
- 新規プロジェクト：2-3個の原則に絞る
- 既存プロジェクト：1個の原則から改善

### Q. ユーザーテストなしで実装してもいい？
**A.** できるだけ避けてください。最低限、以下は確認しましょう：
- [ ] 3-5人のユーザーにタスク完了まで観察
- [ ] 困った瞬間をメモする
- [ ] 改善案を試す

### Q. モバイルとデスクトップで原則を変える？
**A.** 基本は同じですが、**実装方法**は変わります：

| | モバイル | デスクトップ |
|---|---------|-----------|
| **視覚的階層** | より強調 | 適度に |
| **ボタンサイズ** | 44x44px以上 | 30-40px |
| **アニメーション** | 素早く（0.2秒） | やや長く（0.3秒） |
| **余白** | 狭く | ゆったり |

### Q. アクセシビリティと心理学の関係は？
**A.** 密接に関連しています：

```
視覚障害者（スクリーンリーダー）
    ↓
ラベルが必要（認知負荷削減）
    ↓
すべてのユーザーに役立つ
```
